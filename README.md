# resume-template

A resume template, written in LaTeX, that is based off of Harvard's resume template.

## Getting Started

Clone this repository and ensure you have a LaTeX distribution installed on your system.

## How to Install

```bash
git clone https://github.com/<owner>/resume-template.git
cd resume-template
```

## How to Run

Compile the `main.tex` file using:

```bash
pdflatex main.tex
```
